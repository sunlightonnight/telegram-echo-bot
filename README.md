# Telegram Echo Bot

Простой Telegram-бот на Python, который отвечает на любое сообщение.

A simple Telegram bot in Python that replies to any message.

---

## Как запустить / How to run

1. Установи Python 3.x
2. Установи библиотеку: `pip install python-telegram-bot`
3. Скачай `tgbot1.py`
4. Замени `Ваш Токен.` на свой токен от @BotFather
5. Запусти: `python tgbot1.py`
